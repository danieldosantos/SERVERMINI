## Integração de Ferramentas
- **ProSkillAPI:** usar os arquivos `common.yml`, `circle_terra.yml`, `circle_lua.yml` e `circle_tempestade.yml` para registrar as magias e mapear custos/cooldowns. Cada entrada já marca tags de uso para scripts e HUD.
- **MythicMobs:** itens em `plugins/MythicMobs/Items/druid_items.yml` disparam as habilidades e carregam NBT de bloqueio (`class_druid` e subgrupos). Cada tier oferece atalhos para 3–4 skills chave.
- **Skript:** `plugins/Skript/scripts/druid_cmds.sk` fornece aliases (/natureza, /totem, /wildshape, /circulo_*) e kit QA, além da gestão de charges com scoreboard PlaceholderAPI/LuckPerms.
- **LuckPerms:** aplicar grupos `class_druid`, `class_druid.terra`, `class_druid.lua`, `class_druid.tempestade` para liberar habilidades/itens.
- **PlaceholderAPI/Scoreboard:** HUD de Forma Selvagem usa objetivos `druid_charges`, `druid_charges_max` e `druid_ooc` exibidos via action bar.
